# Howell Cho - Personal Portfolio

A modern, responsive personal portfolio website built with React, TypeScript, and Tailwind CSS. This project showcases my journey as a computer science student, featuring my projects, blog posts, certifications, and achievements.

## 🚀 Features

- **Modern Design**: Clean, coffee/Ghibli-inspired aesthetic with warm color palette
- **Responsive Layout**: Optimized for all devices and screen sizes
- **Interactive Components**: Smooth animations and hover effects
- **Multi-Page Navigation**: Home, Diary (Blog), About, Portfolio, Achievements, Contact
- **Demo Content**: Comprehensive demo data showcasing various sections
- **Image Optimization**: Proper image sizing and fallback handling
- **TypeScript**: Full type safety and better development experience

## 🛠️ Tech Stack

- **Frontend**: React 18.3.1
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4.1.12
- **Animations**: Motion (Framer Motion)
- **Icons**: Lucide React
- **Build Tool**: Vite 6.3.5
- **UI Components**: Radix UI primitives

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Howell-Cho319/howell-cho319.github.io.git
   cd howell-cho319.github.io
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
