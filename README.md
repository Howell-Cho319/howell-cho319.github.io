A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS.

## 🚀 Tech Stack

- React 18 with TypeScript
- Tailwind CSS 4
- Vite 6
- Framer Motion
- React Router

## 🔧 Development

### Prerequisites
- Node.js 18+
- npm or yarn

### Setup
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
```
